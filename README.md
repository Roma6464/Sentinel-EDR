# Sentinel-EDR
Prototype d'EDR (Endpoint Detection and Response) modulaire en User-Mode.
Sentinel-EDR est un prototype de solution de détection et de réponse sur les points de terminaison (EDR) modulaire, conçu pour offrir une visibilité approfondie sur l'activité des processus Windows en User-Mode.
Ce projet illustre l'interaction entre le bas-niveau (C), l'analyse de données (Python) et l'interface utilisateur (C#).
